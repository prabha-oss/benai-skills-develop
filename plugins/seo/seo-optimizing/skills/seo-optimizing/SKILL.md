# SEO Optimizing Skill

This skill assesses provided text or URLs and offers actionable recommendations to improve search engine visibility.

## Capabilities

1.  **Content Analysis**: Scan text for keyword density, readability (Flesch-Kincaid), and structure.
2.  **On-Page SEO Checks**: Verify title tags, meta descriptions, H1-H6 hierarchy, and alt text usage.
3.  **Competitor Comparison**: Compare content against top-ranking results for specific keywords (requires external search access).
4.  **Optimization Suggestions**: Rewrite sections to improved keyword targeting without keyword stuffing.

## Usage

Provide the text or URL you want to optimize, along with the target keyword(s).

### Example Request

"Optimize the following blog post for the keyword 'best coffee beans 2024'."
